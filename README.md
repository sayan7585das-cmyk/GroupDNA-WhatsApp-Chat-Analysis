# GroupDNA-WhatsApp-Chat-Analysis
A Python and NumPy-based analysis of WhatsApp group chat data with visualizations and statistical insights.

## Overview

GroupDNA is a data analysis project that explores WhatsApp group chat data using Python. The project extracts useful statistics, identifies activity patterns, and visualizes user interactions.

## Features

- Total message analysis
- User activity statistics
- Most active participants
- Busiest day and hour
- Word frequency analysis
- Emoji usage analysis
- NumPy-based heatmap visualization
- Message timeline analysis

## Technologies Used

- Python
- NumPy
